Graph 1: Full Exoplanet Temperature Map
This scatter plot visualizes 4,526 confirmed exoplanets across two thermal dimensions: the effective temperature of the host star (x-axis) and the planet's calculated equilibrium temperature (y-axis, log scale). Each point is colored by planet mass in Jupiter masses and sized by planet radius in Earth radii. The habitable zone (200–350 K) is shaded for reference.
Graph 2: Habitable Zone Zoom
This graph isolates the 247 exoplanets with equilibrium temperatures between 200 and 350 K — the conventional liquid-water habitable zone boundary — orbiting host stars between 2,400 and 6,600 K. Reference lines mark the warm edge of the habitable zone (350 K) and Earth’s own equilibrium temperature (~255 K).
